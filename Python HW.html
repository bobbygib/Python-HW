About Python

  • High-level programming language
  • Object-oriented programming - designs programs based off of objects interacting with each other(similar to c++, c#, javascript, etc...)
  • Very clear syntax that makes use of whitespace
  • Influenced by ABC programming language
  • Easier to read for humans
  • Large set of rules, know as PEP 8
  • Open source language
  • Uses class/objects, loops, strings, numbers,
  • Most taught language in American schools
  • Can be used on Rasberry Pi

Who invented Python?

  • Guido Van Rossum
  • Born in the Netherlands 1956
  • Worked with Google and currently with Dropbox
  • Currently still oversees Phython development
  • Known as BDFL(benevolent dictator for life)

When and why was Python invented?

  • Developed in the 1980s, imlemented in December 1989
  • Orginally developed for the operating system Amoeba, Phyton was created to use Amoebas system calls that ABC could not
  • First released to the public in February 1991
  • Many more releases since
  • Current new version, 3.6.0
